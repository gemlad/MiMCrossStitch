# MiMCrossStitch
A cross-stitch pattern of a wizard.

The PDF contains a scan of the pattern pages, followed by a scan of the folds, which unfortunately have some missing stitches as the pattern is old!

There are also PNGs of the pages in case that is more helpful.

I am going to attempt to create a proper PDF pattern in WinStitch, but please don't hold your breath! If anyone else is a wizard (pun intented) at converting patterns, please give it a go and I can host it here.

Feel free to transfer these files to other sites - I do not own the copyright to this pattern. It is a discontinued Dimensions pattern. Take down on request of the designer or copyright holder (but please make it available again!)
